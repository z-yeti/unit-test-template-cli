# Quick Template a Unit Test 🧪

## npx:
### Usage ⌨️:
```console
npx unit-tests-template-cli <full_path_to_file> -p
```

## Global:
### Installation 💿:
- clone repo to your code/utils or whatever you wish
- `cd <where_you_cloned>`
- `npm i -g .`
### Usage ⌨️:
```console
unit-tests-template <full_path_to_file> -p
or
utt <full_path_to_file> -p
```
